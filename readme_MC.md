## Semesterarbeit

Die Semesterarbeit muss in dem Ordner [./src](../pittWebPage/src) realisiert werden. 
Eine Grundstruktur zum Trennen von JavaScript und CSS Dateien ist bereits angelegt, 
die HTML Dateien kannst du in das Hauptverzeichnis legen. 

Beispielsweise:

```
...
src/css/format.css
src/scripts/app.js
src/index.html
src/detailview.html
...
```

## Dokumentation und Planung

Wireframes und Beschreibungen zu der Semesterarbeit sind in dem Ordner [./docs](../pittWebPage/docs) zu hinterlegen.

## Lernleistungen

Die Lernleistungen U1 und U2 sind im Moodle abzugeben. 
Die Daten Vorlagen für die Aufgabe U3 befinden sind im Ordner [./xml](xml).
